## Task Requirements

The task asked to build a webpage that includes:

- NGO name
- A short tagline or heading
- About section
- One image
- One button (Volunteer / Join Us / Donate)
- Footer

All required elements are included in this project.

---

## What I Built

A single-page HTML website for Queens Of Change Foundation — a 100% women-led NGO working to empower underprivileged women and girls across India through menstrual health, education, and skill development.

## Features

- **NGO name** in the header
- **Tagline** — *"Empowering women through dignity, health, and opportunity."*
- **About section** describing the foundation's mission and work
- **One image** — a full-width community photo
- **Donate Now button** linking to the foundation's donation page
- **Footer** with name, registration info, and tagline
- Warm colour palette (terracotta, cream, deep brown)
- Google Fonts — Playfair Display + Lora
- Fully responsive layout
